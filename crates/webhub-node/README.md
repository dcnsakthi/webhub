# microsoft-webhub-node

Node.js native addon for the [webhub](https://github.com/microsoft/webhub) framework, built with [napi-rs](https://napi.rs).

## Overview

`microsoft-webhub-node` compiles to a platform-specific `.node` addon that exposes webhub's rendering API to Node.js hosts (e.g. Express, Fastify) without spawning a subprocess.

## Documentation

See the [webhub repository](https://github.com/microsoft/webhub) for full usage guides and examples.
