# microsoft-webhub-expressions

Expression evaluator for the [webhub](https://github.com/microsoft/webhub) framework. Evaluates template binding expressions against JSON state at render time.

## Overview

`microsoft-webhub-expressions` provides a fast, allocation-conscious expression engine for resolving data bindings, conditional expressions, and loop iterators inside webhub templates.

## Documentation

See the [webhub repository](https://github.com/microsoft/webhub) for full usage guides and examples.
